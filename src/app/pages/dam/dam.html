Desarrollo de aplicaciones Multiplataforma (DAM)

El titulado en DAM se encarga de la lógica y la interfaz de programas que no dependen necesariamente de un navegador para funcionar.

1º Desarrollo de Aplicaciones Multiplataforma

  • Programación

    • Bases de la programación con Java: lógica, estructuras de control, POO y buenas prácticas

  • Lenguajes de marcas y sistemas de gestión de información

    • HTML, CSS y XML para crear y estructurar contenido web de forma correcta

  • Bases de datos

    • Diseño de bases de datos y consultas SQL para almacenar información de las aplicaciones

  • Sistemas informáticos

    • Funcionamiento del hardware, sistemas operativos y nociones básicas de redes

  • Entornos de desarrollo

    • Uso de IDEs, control de versiones con Git y trabajo en equipo

  • Itinerario personal para la Empleabilidad I

    • Orientación laboral, habilidades profesionales y primeros pasos en el mundo laboral


2º Desarrollo de Aplicaciones Multiplataforma

  • Programación multimedia y dispositivos móviles

    • Desarrollo de aplicaciones móviles y multimedia, principalmente con Android

  • Desarrollo de interfaces

    • Diseño de interfaces gráficas usables y atractivas

  • Acceso a datos

    • Conexión de las aplicaciones con bases de datos, ficheros y APIs externas

  • Programación de servicios y procesos

    • Desarrollo de servicios, procesos y comunicaciones entre aplicaciones

  • Sistemas de gestión empresarial

    • Uso y personalización de ERPs y CRMs para entornos empresariales

  • Empresa e iniciativa emprendedora

    • Creación de empresas, ideas de negocio y nociones legales y económicas

  • Proyecto de DAM

    • Desarrollo de un proyecto completo aplicando todo lo aprendido

  • Formación en centros de trabajo (FCT)

    • Prácticas en empresa para adquirir experiencia real
