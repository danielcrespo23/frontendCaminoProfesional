¿Qué es DAM?

El Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Multiplataforma te prepara para diseñar y programar aplicaciones que funcionan en múltiples entornos: móvil, escritorio y web.

Aprenderás a desarrollar para Android, iOS y escritorio, integrando bases de datos, APIs y servicios en la nube. El perfil DAM es uno de los más completos y versátiles del mercado tecnológico.

Tecnologías que dominarás:

Java Kotlin Android Swift React Native Flutter .NET SQL Firebase Git

Preguntas Frecuentes

¿Qué diferencia hay entre DAM y DAW?

@if (faqAbierto === 1) {

DAM se centra en aplicaciones nativas para móvil y escritorio (Android, iOS, Windows), mientras que DAW se enfoca en aplicaciones web que se ejecutan en el navegador. DAM usa más Kotlin/Java/Swift, DAW más JavaScript/HTML/CSS.

}

¿Necesito saber programar para empezar?

@if (faqAbierto === 2) {

No es obligatorio. El ciclo empieza desde los fundamentos de programación. Tener conocimientos previos ayuda, pero el temario está diseñado para partir de cero.

}

¿Puedo acceder con bachillerato?

@if (faqAbierto === 3) {

Sí. Puedes acceder con Bachillerato, Grado Medio de FP o superando la prueba de acceso a Grado Superior. También con titulación universitaria.

}

¿Cuáles son las salidas laborales más demandadas?

@if (faqAbierto === 4) {

Las más demandadas son: Desarrollador Android/iOS, Full-Stack Developer y desarrollador de aplicaciones empresariales. El perfil DAM es muy versátil y tiene salida en casi todos los sectores.

}
