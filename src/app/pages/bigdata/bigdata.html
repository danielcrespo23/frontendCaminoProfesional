¿Qué es la Especialización en Big Data?

La Especialización en Big Data y Análisis de Datos está diseñada para titulados en informática que quieren dominar el procesamiento y análisis de volúmenes masivos de datos para generar valor de negocio.

Aprenderás a construir pipelines de datos, usar el ecosistema Hadoop/Spark y crear visualizaciones que convierten datos brutos en insights estratégicos para las empresas.

Tecnologías que dominarás:

Python Apache Spark Hadoop MongoDB Kafka Tableau Power BI AWS SQL avanzado Airflow

"Se generan 2,5 quintillones de bytes de datos al día — alguien tiene que procesarlos"
