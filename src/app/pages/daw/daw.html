¿Qué es DAW?

El Ciclo Formativo de Grado Superior en Desarrollo de Aplicaciones Web te prepara para ser un profesional capaz de diseñar, desarrollar y mantener aplicaciones web modernas utilizando las tecnologías más demandadas del mercado.

Aprenderás tanto el frontend (lo que ve el usuario) como el backend (la lógica del servidor), convirtiéndote en un desarrollador full-stack.

Tecnologías que dominarás:

HTML5 CSS3 JavaScript React Angular PHP Java MySQL MongoDB Git

Preguntas Frecuentes

¿Qué requisitos necesito para acceder a DAW?

@if (faqAbierto === 1) {

Necesitas tener Bachillerato, un Grado Medio de FP, o haber superado la prueba de acceso a Grado Superior. También puedes acceder con titulación universitaria.

}

¿Necesito saber programar antes de empezar?

@if (faqAbierto === 2) {

No, el ciclo empieza desde cero. Aprenderás programación desde los fundamentos hasta nivel profesional. Aunque ayuda tener conocimientos previos, no es obligatorio.

}

¿Puedo hacer el ciclo en modalidad dual?

@if (faqAbierto === 3) {

Sí, ofrecemos modalidad dual donde combinas clases con trabajo remunerado en empresas desde el primer año. Es una excelente forma de aprender y ganar experiencia al mismo tiempo.

}

¿Cuál es la diferencia entre DAW y DAM?

@if (faqAbierto === 4) {

DAW se enfoca en aplicaciones web (navegadores), mientras DAM se centra en aplicaciones multiplataforma (móviles, escritorio). DAW usa más HTML/CSS/JavaScript, DAM usa más Java/Kotlin para Android.

}
