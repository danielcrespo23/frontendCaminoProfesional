Desarrollo de aplicaciones Web (DAW)

El titulado en DAW se encarga de crear, implementar y mantener aplicaciones.

1º Desarrollo de Aplicaciones Web

  • Programación

    • Fundamentos de programación con Java: lógica, POO y bases del desarrollo web

  • Lenguajes de marcas y sistemas de gestión de información

    • HTML, CSS, XML y JSON para crear y estructurar páginas web

  • Bases de datos

    • Diseño de bases de datos y consultas SQL para aplicaciones web

  • Sistemas informáticos

    • Hardware, sistemas operativos y nociones básicas de redes

  • Entornos de desarrollo

    • Uso de IDEs, Git y trabajo colaborativo en proyectos web

  • Itinerario personal para la Empleabilidad I

    • Orientación laboral y habilidades profesionales

  • Módulo profesional optativo

    • Asignatura adicional para ampliar conocimientos web


2º Desarrollo de Aplicaciones Web

  • Desarrollo web en entorno servidor

  • Diseño de interfaces web

  • Despliegue de aplicaciones web

    • Publicación de webs y aplicaciones en servidores reales

  • Empresa e iniciativa emprendedora

    • Creación de empresas, ideas de negocio y mundo laboral
