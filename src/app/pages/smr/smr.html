¿Qué es SMR?

El Ciclo Formativo de Grado Medio en Sistemas Microinformáticos y Redes es el punto de entrada ideal al mundo de la informática. Aprenderás a instalar, configurar y mantener equipos y redes de área local.

Es la base perfecta para incorporarte al mercado laboral como técnico de soporte y, si quieres seguir creciendo, acceder después a ciclos superiores como ASIR, DAW o DAM.

Tecnologías que aprenderás:

Windows Linux Cisco básico TCP/IP Office 365 Hardware PC Wi-Fi Impresoras Ticketing Virtualización

Preguntas Frecuentes

¿Qué requisitos necesito para acceder a SMR?

@if (faqAbierto === 1) {

Necesitas el título de ESO (Educación Secundaria Obligatoria) o equivalente. También puedes acceder superando la prueba de acceso a Grado Medio si tienes 17 años.

}

¿Puedo acceder a ASIR o DAW después de SMR?

@if (faqAbierto === 2) {

Sí. Con el título de SMR (Grado Medio) puedes acceder directamente a cualquier Grado Superior de FP como ASIR, DAW o DAM sin necesidad de Bachillerato.

}

¿Necesito conocimientos previos de informática?

@if (faqAbierto === 3) {

No. SMR está diseñado para empezar desde cero. Aprenderás desde cómo funciona un ordenador hasta configurar redes pequeñas. Sólo necesitas ganas de aprender.

}

¿Cuánto dura el ciclo y cuándo son las prácticas?

@if (faqAbierto === 4) {

El ciclo dura 2 años. Las prácticas en empresa (FCT) son aproximadamente 380 horas y se realizan al final del segundo curso, generalmente de marzo a junio.

}
