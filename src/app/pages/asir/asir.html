¿Qué es ASIR?

El Ciclo Formativo de Grado Superior en Administración de Sistemas Informáticos en Red forma profesionales capaces de instalar, configurar y mantener toda la infraestructura tecnológica de una empresa.

Desde servidores y redes hasta cloud computing y ciberseguridad, el técnico ASIR es el pilar que mantiene los sistemas funcionando 24/7. Un perfil imprescindible en cualquier organización.

Tecnologías que dominarás:

Linux Windows Server Cisco VMware Docker AWS Azure Active Directory Bash Python

Preguntas Frecuentes

¿Qué requisitos necesito para acceder a ASIR?

@if (faqAbierto === 1) {

Necesitas Bachillerato, un Grado Medio de FP, o haber superado la prueba de acceso a Grado Superior. También puedes acceder con titulación universitaria.

}

¿Cuál es la diferencia entre ASIR y SMR?

@if (faqAbierto === 2) {

SMR es Grado Medio (técnico) y se centra en soporte y mantenimiento básico. ASIR es Grado Superior (técnico superior) y abarca administración avanzada de servidores, redes complejas, cloud y seguridad. Tiene más profundidad técnica y mejores salidas salariales.

}

¿ASIR tiene salidas en cloud y AWS?

@if (faqAbierto === 3) {

Sí, el perfil ASIR es muy demandado en empresas cloud. La formación en virtualización, Docker y servicios de red es la base perfecta para certificarse en AWS, Azure o Google Cloud tras el ciclo.

}

¿Puedo hacer ASIR en modalidad online?

@if (faqAbierto === 4) {

Sí, muchos centros ofrecen ASIR en modalidad semipresencial o completamente online. Las prácticas en empresa (FCT) sí requieren presencialidad en la empresa asignada.

}
